## 9.0.0
**リリース日：** 2026-01-30 · Xcode 26.2 · Swift 6.x · iOS 14 以上

- **新機能：** 改善されたトークン管理のための `MissionTokenProvider` プロトコル
  - SDK がトークン期限切れを自動処理 — 手動での `.tokenExpired` 処理が不要に
  - SDK API 呼び出し前に `.online` ステータスを待つ必要がなくなりました
  - `initSdk(appCode:tokenType:tokenProvider:)` が手動の `startSession` 呼び出しを置き換えます
  - `startSession` API は引き続き動作しますが、自動トークン管理の恩恵はありません
- Xcode 26.2 でビルド

アップグレード手順は [v9 移行ガイド](migration-to-v9.md) をご確認ください。
